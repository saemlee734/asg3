Zombie Run: Save the Bunny from the Runes!
Sabrina Lee | saemlee@ucsc.edu
Build a shelter for the lost bunny before the zombie catches it in the cursed runes. Each block you place brings the zombie one step closer. Every run is different!

Game Controls

  • Left click - Place blocks (build the bunny's home)
  • Right click - Remove blocks
  • WASD/Arrow keys - Move around
  • Q/E - Look left/right
  • Click canvas - Enable mouse look

World Features

  • Turns sunset at 4 blocks placed
  • Turns night at 8 blocks placed
  • Reset World if the task isn't achieved
  • Each block placed causes the zombie to move one step closer to the bunny
  • Structures and world elements are randomly generated for unique playthroughs
Please use a browser that supports "canvas"
